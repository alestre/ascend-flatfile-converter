# Flatfile Converter

Convert between JSON, CSV, XML, YAML and TOML - instantly, privately, for free.

Everything runs locally in your browser (pure JS parsers/serializers, no dependencies). No upload, no server, no tracking, no ads. Works fully offline once loaded.

## Usage

Download [`flatfile-converter.html`](./flatfile-converter.html) and open it in any modern browser. That's it - no install, no build step, no internet connection required.

- Paste data or drop a file - format is auto-detected (override via the dropdown)
- CSV gets a settings modal for delimiter, quoting, escaping, header handling, and nested-data strategy, on both input and output
- A live preview (first 3 rows) shows the effect of CSV settings before converting
- Copy to clipboard or save the result as a file
- Built-in format reference cards (JSON, CSV, XML, YAML, TOML) with spec links and common gotchas

## Online version

A hosted, always up-to-date version of this tool is available at [ascend-digital.net/tools/flatfile](https://ascend-digital.net/tools/flatfile/), available in many other languages, alongside other privacy-first browser tools (image conversion, steganography, and more).
